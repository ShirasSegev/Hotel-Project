# Hotel-Project
web-based hotel room reservation
built as part of an educational project to practice software architecture, modular design, and full-stack web development principles.
The system enables users to browse rooms, check availability, and make reservations through a structured and interactive interface.

*Features*
  -Room catalog with detailed views
  -Date-based availability checking
  -Booking and reservation flow
  -Form validation and user feedback

*Tech Stack*
  -Frontend: JavaScript, HTML, CSS
  -Architecture: Layered structure separating UI, business logic, and data handling
